✅ 레지스터(Register)란?

레지스터는 CPU 내부에 있는 아주 작고 빠른 기억장치입니다.
주된 역할은 데이터를 임시로 저장하고 처리하는 것이에요.

🧠 왜 필요한가요?

CPU는 연산을 매우 빠르게 처리합니다. 하지만 **메인 메모리(RAM)**는 상대적으로 느리기 때문에, CPU가 바로 데이터를 읽고 쓰기에는 비효율적입니다.
그래서 레지스터에 데이터를 잠깐 저장해두고, CPU가 직접 빠르게 접근할 수 있도록 하는 거예요.

📦 레지스터의 특징
특징		설명
작다		저장 용량이 매우 작습니다. 보통 수십 개 정도 있어요.
빠르다	CPU가 가장 빠르게 접근할 수 있는 기억장치입니다.
휘발성	전원이 꺼지면 내용이 사라집니다.
비싸다	제조 비용이 높기 때문에 적은 수만 탑재됩니다.

🧮 레지스터의 종류
누산기(Accumulator)	연산 결과를 임시로 저장
데이터 레지스터(Data Register)	연산에 사용할 데이터를 저장
주소 레지스터(Address Register)	메모리의 주소를 저장
명령어 레지스터(Instruction Register, IR)	현재 실행 중인 명령어를 저장
프로그램 카운터(PC)	다음에 실행할 명령어의 주소를 저장
상태 레지스터(Flags Register)	연산 결과의 상태(예: carry, zero 등)를 저장
